KnitLens 개인정보 처리방침

최종 업데이트: 2026-05-09

시행일: 출시일 (App Store/Play Store 등록일)

연락처: chil9199@gmail.com

KnitLens (이하 "본 앱") 은 「개인정보 보호법」 및 관련 법령에서 정한 사항을 준수하여 사용자의 개인정보를 안전하게 처리합니다.

1. 수집하는 개인정보 항목

1-1. 회원가입 시 (선택)

  • 이메일 주소
  • 비밀번호 (단방향 암호화 저장)
  • 이름 (선택)

1-2. 서비스 이용 시 자동 수집

  • 사용자가 촬영/업로드한 사진 — 도안 생성 분석 목적
  • 생성된 도안 데이터 (재질, 게이지, 부위 정보 등) — 보관함 저장
  • 앱 사용 통계 — 크래시 정보, 오류 발생 시점 (Sentry)
  • 기기 정보 — OS 버전, 앱 버전 (디버깅 목적)

1-3. 결제 시 (Pro 가입자, 향후 출시)

  • App Store / Google Play 의 결제 영수증 (영수증 검증용)
  • 본 앱은 신용카드 정보를 직접 수집하지 않습니다 — Apple/Google 처리

1-4. 수집하지 않는 항목

  • 사용자의 위치 정보
  • 연락처 / 주소록
  • 마이크 / 음성 데이터
  • 다른 앱 사용 기록

2. 개인정보 이용 목적

| 목적 | 사용 항목 |

|---|---|

| 도안 생성 (핵심 기능) | 촬영/업로드 사진 |

| 보관함 (도안 저장 / 복원) | 이메일, 도안 데이터 |

| 회원 인증 / 계정 관리 | 이메일, 비밀번호, 이름 |

| 서비스 개선 (크래시 진단) | 크래시 정보, 기기 정보 |

| Pro 결제 검증 (향후) | 결제 영수증 |

3. 사진 데이터 처리 — 핵심 안내

본 앱은 사용자 사진을 외부 AI 서비스 (Google Gemini Vision API) 로 전송하여 사물 형태를 분석합니다.

사진 처리 흐름

1. 사용자가 카메라/갤러리에서 사진 선택

2. 사진을 Google Gemini Vision API 로 일시 전송 (HTTPS 암호화)

3. AI 가 사물의 형태/색상/비율을 분석해 텍스트 형태의 데이터 (JSON) 반환

4. 본 앱이 분석 결과로 도안 생성

5. 사용자 사진 원본은 본 앱 서버에 저장되지 않습니다

6. 사용자 사진은 Google Gemini API 의 모델 학습에 사용되지 않습니다 (Google Cloud Vertex AI 정책)

사진 데이터 보관

  • 본 앱 서버: 저장 X
  • 사용자 기기: 사용자가 명시적 저장 (보관함) 시에만 도안과 함께 보관
  • Google API: API 호출 처리 직후 폐기 (Vertex AI 정책)

5. 개인정보 제3자 제공

본 앱은 다음 외부 서비스에 한해 사용자 데이터를 처리 위탁합니다.

| 서비스 | 위탁 항목 | 위탁 사유 |

|---|---|---|

| Google Gemini API (Google LLC) | 사진 (분석 직후 폐기) | AI 도안 분석 |

| Sentry (Functional Software Inc.) | 크래시/오류 정보 | 서비스 안정성 |

| Apple App Store | 결제 영수증 (Pro) | 결제 처리 |

| Google Play Store | 결제 영수증 (Pro) | 결제 처리 |

| Railway (서버 호스팅) | 도안 데이터 | 보관함 서비스 |

| Supabase (DB) | 회원 정보, 도안 데이터 | 데이터베이스 |

| AdMob (Google) — 향후 출시 | 광고 ID, 기기 정보 | 광고 노출 |

본 앱은 위 외에 사용자 동의 없이 제3자에게 개인정보를 제공하지 않습니다.
